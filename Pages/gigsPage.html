GIG: Pong

We needed to make a remake for pong with our own twist to it. So i was planning to make the ball a friends head the ball and in a picture further down you can see that. And i also made the sticks you play with trashcans and at last i edited the background. This was the first ever project we had to make for school.

It was made on a custom engine that school made it was called canvas engine but we did use c# for it to code. It was also the first ever time i had programmed in my life except for the entice test we needed to make to get accepted in the school.

But it was very fun to learn about the details of c# and it was cool to learn a new language. I sadly don't have any more scripts of this project. I only have the screenshot down below.

GIG: Black Jack

Black jack is my first ever web game i made with only JavaScript and a bit of html and css. It was for my internship i had at Valtech a web developer company. It was a super fun project to learn about a new programming language.

It was made after I had a few months of learning about programming languages like: Javascript, React, CSS, HTML and some others. It was a lot of learning but it was pretty nice to have the knowledge of those languages to then test and see if you can make a small project that will prove that you understand the basic's.

I was also happy with the end result of the project. It was not perfect but it was working so you know thats what i like to see. If you want to see the end result or try the game: Play here
